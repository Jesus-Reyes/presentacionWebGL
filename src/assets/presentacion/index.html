¿Que es webGL?

API de Javascript para poder comunicarse con la GPU del navegador

    Caracteristicas

  • Javascript en monohilo (una cosa a la vez, asincrono; sincrono en ejecuacion )
  • THREE pinta triangulos
  • GPU se hace en un hilo separado
  • Especializada en operaciones con matrices "Pinta triangulos "

Terminología

    Shaders

  • Vertex Shader

    • Coordenadas, vertices, posiciones

  • Cortex Shader

    • textura, luces, sombras

Diferencias WEBGL vs THREE.JS

  • Definir los shader cortex y vertex shader
  • control total de la gpu
  • Mejor rendimiento
  • Clases y metodos que infieren los shader
  • Control de formas, escenas, camaras, vistas
  • No se tiene el control total de la GPU
